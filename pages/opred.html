Главная страница Определения

Из определения синуса следует, что -1 ⩽ sinα ⩽ 1. Поэтому если |α| > 1, то уравнение sinx = α не имеет корней. Например, уравнение sinx = 2 не имеет корней.

Арксинусом числа α ∈ [-1;1] называется такое число α ∈ [-π/2, π/2], синус которого равен α:
arcsina = α, если sinα = a и -π/2 ⩽ α ⩽ π/2.

Таким образом, все корни уравнения sinx = α, где |α| ⩽ 1, можно находить по формуле:
x = (-1)^n acrsinα + πn, где n ∈ Z.

Частные случаи:
sinx=0, x = πn, n ∈ Z.
sinx=1, x = π/2 + 2πn, n ∈ Z.
sinx=-1, x = -π/2 + 2πn, n ∈ Z.

Так же можно доказать, что для любого значения α ∈ [-1; 1] справедлива следующая формула:
arcsin(-α) = -arcsinα

Из определения косинуса следует, что -1 ⩽ cosα ⩽ 1. Поэтому если |α| > 1, то уравнение cosx = α не имеет корней. Например, уравнение cosx = -1,5 не имеет корней.

Арккосинусом числа α ∈ [-1;1] называется такое число α ∈ [0,π], косинус которого равен α:
arccosa = α, если cosα = a и 0 ⩽ α ⩽ π.

Таким образом, все корни уравнения cosx = α, где |α| ⩽ 1, можно находить по формуле:
x = ±arccosα + 2πn, где n ∈ Z.

Частные случаи:
cosx=0, x = π/2 + πn, n ∈ Z.
cosx=1, x = 2πn, n ∈ Z.
cosx=-1, x = π + 2πn, n ∈ Z.

Так же можно доказать, что для любого значения α ∈ [-1; 1] справедлива следующая формула:
arccos(-α) = π - arccosα

Из определения тангенса следует, что tgx может принимать любое действительное значение. Поэтому уравнение tgx=α имеет корни при любом значении α.

Арктангенсом числа α ∈ R называется такое число α ∈ [-π/2, π/2], тангенс которого равен α:
arctga = α, если tgα = a и -π/2 < α < π/2

Таким образом можно сказать, что все корни уравнения tgx = α, где α ∈ R, выражаются следующей формулой:
x = arctgα+πn, n ∈ Z.

Так же можно доказать, что для любого значения α ∈ R справедлива следующая формула:
arctg(-α) = -arctgα

Тригонометрический круг — это самый простой способ начать осваивать тригонометрию. Он легко запоминается, и на нём есть всё необходимое.
Тригонометрический круг заменяет десяток таблиц.

Вот что мы видим на этом рисунке:
1. Перевод градусов в радианы и наоборот. Полный круг содержит 360 градусов, или 2π радиан.
2. Значения синусов и косинусов основных углов. Помним, что значение косинуса угла мы находим на оси X, а значение синуса — на оси Y.
3. И синус, и косинус принимают значения от -1 до 1.
4. Значение тангенса угла α тоже легко найти — поделив sinα на cosα. А чтобы найти котангенс — наоборот, косинус делим на синус.
5. Знаки синуса, косинуса, тангенса и котангенса.
6. Синус — функция нечётная, косинус — чётная.
7. Тригонометрический круг поможет увидеть, что синус и косинус — функции периодические. Период равен 2π.
